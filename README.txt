NAppGUI is an SDK to develop software projects, that work on any desktop platform (Windows, macOS or Linux), using the C programming language. C++ is allowed, but not indispensable. 

You must use CMake and the standard compilers on each platform (VisualStudio, Xcode or GCC).

For more information about building process visit:
https://nappgui.com/en/start/quick.html
